# 📝 Todo App (JavaScript)

A simple and clean Todo List application built using **HTML, CSS, and JavaScript** with Local Storage support.

---

## 🚀 Features

- ➕ Add new todos
- ❌ Delete todos
- 💾 Save data using Local Storage (data stays after refresh)
- 📅 Add due date for tasks
- 🖥️ Simple and responsive UI

---

## 🛠️ Tech Stack

- HTML
- CSS
- JavaScript (Vanilla JS)
- Local Storage API
